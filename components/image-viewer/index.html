ImageViewer 图片查看器 #

用于全屏预览图片的组件。

基础用法 #

最基础的用法，点击按钮弹出图片预览。

多图预览 #

支持多张图片切换预览。

vue
<template>
  <cs-button @click="show = true">点击预览图片</cs-button>
  <cs-image-viewer 
    v-if="show" 
    :url-list="urlList" 
    @close="show = false" 
  />
</template>

<script setup>
import { ref } from 'vue'
const show = ref(false)
const urlList = [
  'https://fuss10.elemecdn.com/a/3f/3302e58f9a181d2509f3dc0fa68b0jpeg.jpeg'
]
</script>

Attributes #

参数说明类型可选值默认值
url-list预览链接列表string[]—[]
z-index预览层级number—2000
initial-index初始预览图像索引number—0
infinite是否可以无限循环预览boolean—true
hide-on-click-modal是否可以通过点击遮罩层关闭boolean—false
teleported是否将组件插入 bodyboolean—true
close-on-press-escape是否可以通过按下 ESC 关闭boolean—true
zoom-rate缩放速率number—1.2
min-scale最小缩放比例number—0.2
max-scale最大缩放比例number—7
rotate是否显示旋转按钮boolean—true

Slots #

插槽名说明
default自定义内容
close-icon自定义关闭图标
prev-icon自定义上一张图标
next-icon自定义下一张图标

Events #

事件名说明回调参数
close关闭预览时触发—
switch切换图片时触发(index: number)
rotate旋转图片时触发(deg: number)
